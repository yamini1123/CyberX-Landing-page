# CYBER-X Landing Page

Futuristic cyberpunk-inspired landing page designed and developed by using HTML and CSS. This project comprises of neon inspired interface design, responsiveness, glow effects, and modern layouts that work well for portfolio pages, startup pages, technology based products and projects for hackathons.
-Features

* Futuristic cyberpunk theme
* Neon glow effects
* Responsive design
* Interactive hover effects
* Full screen hero section
* Modern feature cards
* Google Fonts support (Orbitron)

-Technologies Used

* HTML5
* CSS3
* Google Fonts

 -File Structure:
index.html
style.css
script.js
README.md

-Steps To Run

1. Clone the repository:
   git clone https://github.com/your-username/cyber-x-landing-page.git
2. Navigate into the project directory.
3. Open index.html in your web browser.
